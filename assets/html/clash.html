自动抓取tg频道、订阅地址、公开互联网上的ss、ssr、vmess、trojan节点信息，聚合去重后提供节点列表

Clash

    配置文件

  • Clash配置文件: {{ .request }}://{{ .domain }}:{{ .port }}/clash/config 一键导入
  • 本地运行时Clash配置文件: http://127.0.0.1:{{ .port }}/clash/localconfig 一键导入

    • Provider

  • Clash proxy-provider：{{ .request }}://{{ .domain }}:{{ .port }}/clash/proxies
  • 筛选代理类型：{{ .request }}://{{ .domain }}:{{ .port }}/clash/proxies?type=ss,ssr,vmess
  • 筛选速度：{{ .request }}://{{ .domain }}:{{ .port }}/clash/proxies?speed=0
  • 筛选国家：{{ .request }}://{{ .domain }}:{{ .port }}/clash/proxies?c=HK,TW,US&nc=JP (c为需要的国家，nc为不需要的国家)
  • 筛选中转：{{ .request }}://{{ .domain }}:{{ .port }}/clash/proxies?filter=r (r为中转，p为pool，rp为中转+pool，nr为非中转，np为非pool，nrp为原生ip)
  • 更新方法：
    • Clash for Windows: 更新配置文件即可
    • ClashX: Dashboard中，点击每个策略组右上角的刷新图标
    • ClashXPro: Config → Update External Sources
